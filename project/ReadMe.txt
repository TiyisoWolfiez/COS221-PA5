//////////////////Navigating this project is as follows/////////////////////

1. Backend contains all the php code that connects and talks to the database
    i) Config contains the logic for connecting to the database
    ii) Api contains all the code for talking to the database itself
2. Frontend contains all the pages that the user views when they visit the website
    i) Assets contains all the assets a user see's when they visit the page, eg images, svg, videos etc.
    ii) Client contains all the client code that talks to the backend
    iii) Pages contains the actual pages a user sees when they visit the website
    iv) Styles contains the stylings for the web pages
